Folien mit Marp in VS Code

marp.app ist ein Open Source (Elektron/Web) Markdown Presentation Framework.

  • VS Code Plugin
  • MathJax Support
  • Syntax/Doku
  • Source Code

Marp funktioniert gut um die Folien "runter zu schreiben". Einfaches Tuning der Darstellung ist ohen großen Aufwand möglich.

Ich werde es wohl in Zukunft für die ersten Drafts verwenden.

Alternativ: Slides.com (Reveal.js)
